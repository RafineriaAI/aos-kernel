# Public Boundary

The public AOS Kernel repository is a curated demonstrator of the kernel control
pattern.

It includes:

- deterministic PASS/WARN/BLOCK decision logic;
- replayable evidence examples;
- Trusted Output v0 unsigned wrapper verification;
- abstract Lean theorem surface;
- integrity manifest verification.

It excludes:

- production policies;
- calibration logic;
- customer integrations;
- official signing keys and signer service;
- enterprise deployment hardening;
- commercial scoring and private workflows.
