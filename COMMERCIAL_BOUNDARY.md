# Commercial Boundary

This repository is a public, source-available proprietary demonstrator of AOS.
It is not an open-source release and does not grant rights to copy, modify,
distribute, commercialize, or create derivative works without written permission.

## Clone-And-Wrap Boundary

Cloning for inspection does not grant permission to repackage, host, sell,
benchmark as a product substitute, wrap as a competing service, or create a
commercial derivative from this demonstrator. The public repository is designed
to show executable kernel fit while withholding private production policies,
calibration, customer adapters, managed-service internals, and commercial
implementation strategy.

## Public Scope

The public repository includes:

- a minimal model-agnostic AOS Kernel surface;
- CLI commands for `demo`, `verdict`, `trust`, and `validate`;
- committed sanitized evidence artifacts;
- integrity verification;
- selected abstract Lean proof surface;
- examples that show the kernel evidence shape.

## Private Scope

The public repository does not include private production material such as:

- full commercial policy logic;
- private calibration or scoring methods;
- enterprise deployment hardening;
- customer-specific integrations;
- customer workflow data;
- private benchmark results;
- support, SLA, or managed-service architecture;
- production key management, signing, or operational controls.

## Intended Reading

The repository is intended to show that the AOS Kernel control pattern is
executable, replayable, and technically inspectable. Commercial evaluation,
private pilots, licensing, and enterprise integration require separate
permission and separate technical material.
